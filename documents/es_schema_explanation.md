# Elasticsearch 索引结构说明

本文件为 **课题三：风险预警知识图谱构建与应用** 的核心数据结构说明，旨在明确 Elasticsearch 中 `t_behavior` 和 `t_regulation` 索引的详细字段定义、类型和约束。
本文档适用于 **课题二：风险指标的智能识别与关联分析** 和 **课题三任务一：风险知识的抽取与表示** 的开发人员，作为数据输入、处理和查询的统一标准和参考依据。

## `t_behavior` 索引

`t_behavior` 索引用于存储已发生的特定行为或事件的信息。

| 属性 | 说明 |
| :--- | :--- |
| `id` | 行为记录的唯一标识符 (Keyword)。 |
| `project_id` | 该行为所属项目的ID (Long)。 |
| `description` | 行为的详细文本描述。该字段会进行分词用于全文搜索 (Text)。 |
| `type` | 行为的类型。可选值包括: "定性", "定量" (Keyword)。 |
| `dimension` | 行为所属的风险维度。可选值包括: "企业关联方风险", "产品合规风险", "劳务合规风险", "企业信用风险", "企业国际合作风险", "供应链风险" (Keyword)。 |
| `tags` | 用于行为分类的标签。可选值包括: "财务指标", "资金使用", "财务报告", "成本控制", "收入增长", "流动性", "盈利能力", "偿债能力", "营运能力", "发展能力", "合同签署", "交易管理", "投资管理", "市场营销", "产品销售", "运营效率", "客户关系管理", "供应商管理", "效率指标", "人力资源", "人事管理", "薪酬管理", "员工发展", "组织效能", "风险管理", "风险管控", "内部举报处理", "安全指标", "质量管控", "法规遵循", "许可证管理", "内控制度执行", "审计管理", "信息披露", "股东大会", "治理结构", "政府关系维护", "监管配合", "客户满意度", "市场表现", "可持续发展", "ESG指标", "社会责任履行", "公益活动", "环境保护", "技术创新", "产品创新", "数字化转型", "研发投入", "合规培训", "制度建设", "流程优化", "标准化管理" (Keyword)。 |
| `status` | 行为的当前状态。可选值包括: "已完成", "进行中", "暂停", "终止" (Keyword)。 |
| `quantitative_data` | 与行为相关的定量数据（如果行为是定量的话）(Double)。 |
| `behavior_date` | 行为发生的日期 (Date)。 |
| `vector` | 行为描述的768维密集向量表示，用于语义相似度搜索 (Dense Vector)。 |
| `created_at` | 行为记录创建时的时间戳 (Date)。 |

## `t_regulation` 索引

`t_regulation` 索引用于存储法律、法规和合规文件的信息。

| 属性 | 说明 |
| :--- | :--- |
| `id` | 法规的唯一标识符 (Keyword)。 |
| `name` | 法规的名称或标题。该字段会进行分词用于全文搜索 (Text)。 |
| `type` | 法规的类型。可选值包括定性、定量(Keyword)。 |
| `dimension` | 法规相关的风险维度。可选值包括: "企业关联方风险", "产品合规风险", "劳务合规风险", "企业信用风险", "企业国际合作风险", "供应链风险" (Keyword)。 |
| `industry` | 法规适用的行业。可选值包括: "银行业", "证券业", "保险业", "信托业", "基金业", "期货业", "金融租赁", "汽车制造", "电子制造", "机械制造", "化工制造", "纺织制造", "食品制造", "医药制造", "钢铁制造", "有色金属", "建材制造", "软件开发", "互联网", "电信运营", "数据服务", "人工智能", "云计算", "石油天然气", "电力", "煤炭", "新能源", "核能", "水电", "房地产", "建筑工程", "装饰装修", "物业管理", "零售业", "批发业", "电子商务", "连锁经营", "航空运输", "铁路运输", "公路运输", "水路运输", "物流仓储", "医疗服务", "医疗器械", "生物技术", "健康管理", "教育服务", "文化传媒", "出版印刷", "广告营销", "农业", "林业", "牧业", "渔业", "农产品加工", "公共事业", "环保服务", "咨询服务", "法律服务", "会计服务", "综合类" (Keyword)。 |
| `tags` | 用于法规分类的标签。可选值包括: "财务指标", "资金使用", "财务报告", "成本控制", "收入增长", "流动性", "盈利能力", "偿债能力", "营运能力", "发展能力", "合同签署", "交易管理", "投资管理", "市场营销", "产品销售", "运营效率", "客户关系管理", "供应商管理", "效率指标", "人力资源", "人事管理", "薪酬管理", "员工发展", "组织效能", "风险管理", "风险管控", "内部举报处理", "安全指标", "质量管控", "法规遵循", "许可证管理", "内控制度执行", "审计管理", "信息披露", "股东大会", "治理结构", "政府关系维护", "监管配合", "客户满意度", "市场表现", "可持续发展", "ESG指标", "社会责任履行", "公益活动", "环境保护", "技术创新", "产品创新", "数字化转型", "研发投入", "合规培训", "制度建设", "流程优化", "标准化管理" (Keyword)。 |
| `region` | 法规适用的地理区域。可选值包括中国、美国、欧盟、多地区(Keyword)。 |
| `applicable_subject` | 法规适用的实体或个人类型。可选值包括政府机构官员、国有企业、关键供应商/承包商、客户、员工、个人用户、公众 (Keyword)。 |
| `full_text` | 法规的完整文本。该字段会进行分词用于全文搜索 (Text)。 |
| `direction` | 指示法规是鼓励性的还是限制性的。可选值包括禁止、必须、中性、允许偏向、禁止偏向 (Keyword)。 |
| `quantitative_indicator` | 法规中提到的定量值或阈值 (Double)。 |
| `vector` | 法规内容的768维密集向量表示，用于语义搜索 (Dense Vector)。 |
| `created_at` | 法规记录创建时的时间戳 (Date)。 |
